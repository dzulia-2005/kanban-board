# Kanban Board

## Description
Kanban board application for managing inquiries with drag & drop functionality.

## Tech Stack
- Next.js
- React
- TypeScript
- @dnd-kit
- TanStack Query
- Ant Design
- dayJS
- react useContext hook

## Features
- Kanban columns by inquiry phase
- Drag & Drop between columns
- Filtering inquiries
- Inquiry details modal
- Mock API with delayed responses

## Setup
```bash
npm install
npm run dev
